# Neomanitai · First Words

**Name That Moment** — a single-file, zero-tracking web tool presenting descriptive terms for observable phenomena in human–AI interaction.

Part of the **Neomanitai research framework** (umbrella for AUGMANITAI, ROBMANITAI, PFT-MKI).

## Status

Demo mode: the tool currently ships with 7 placeholder sample terms to demonstrate the interface. The full term collection will be added incrementally via `terms.json`. The demo banner makes the placeholder status explicit.

## Ethical framework

All use is governed by the bilingual §1 – §26 ethical disclaimer shown in the first-visit gate and accessible via the footer at any time. Key points:

- Purely descriptive linguistic research — no medical, therapeutic, diagnostic, legal, moral or financial claims.
- No instruction, no advice, no offer.
- Living research resource, continuously expanded.
- Misuse — manipulation, deception, surveillance, dual-use — is explicitly excluded and condemned by the author.
- Intended exclusively for users 18 years or older.
